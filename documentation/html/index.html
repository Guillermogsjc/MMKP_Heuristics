MMKP Heuristics

This directory contains source for testing heuristics on the Multiple-Choice Multidimensional Knapsack problem (MMKP).

Supported problem instances are the legacy, 13 problem, Or-Lib library and the MMKP library of 370 problems by Hiremath and Hill.
