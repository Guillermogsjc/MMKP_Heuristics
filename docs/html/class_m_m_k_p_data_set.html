float MMKPDataSet::getVariableConstraintUseDiff ( MMKPSolution &  solution)

Return the percent difference of contraint use, comparing a solution and resources.

Calculated using only the constraints which are violated in a solution. Additionally, the input solution must be multiple-choice feasible.
