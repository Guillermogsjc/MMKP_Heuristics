9  * TLBO MMKP Problem

11  * ::Tlbo parameters::
12  *
13  * multipleChoiceMod:
14  * FixedMaxSurrogate/0:
15  * If one item is selected in a class, continue. If more than one
16  * item selected, choose the one with highest v/[E_r%/n] value. If
17  * none selected, choose item from class with highest v/[E_r%/n].
18  *
19  * FixedMaxRandomSurragate/1:
20  * Same as 0, except if no items are selected, then randomly choose
21  * an item from the class.
22  *
23  * HighProfit/2:
24  * If one item is selcted in a class, continue. If more than one
25  * item is selected choose the one with the highest profit. If
26  * none are selected choose the item in the class which
27  * maximizes profit.
28  *
29  *
30  * multipleDimMod:
31  * FixedMaxSurrogate/0:
32  * Use n, max diff, fixed size surrogate constraint to obtain
33  * multiple dimension feasibility, where n is the
34  * number of constraints of a problem. Additionally, maximum
35  * difference of surrogate values used to quickly obtain
36  * feasbility.
37  *
38  * VariableMaxSurrogate/1:
39  * Use variable, max diff, surrogate constraint to obtain
40  * multiple dimension feasbility. Similar to (0), the max
41  * difference of surrogate values are used to quickly
42  * obtain feasilibty. However, this method calculates
43  * it's surrogate constraint values based not on all problem
44  * consraints, but only the ones violated.
45  *
46  * VariableMinSurrogate/2:
47  * Similar to #1, this method uses a variable size surrogate
48  * constraint equal to the number of violated constraints.
49  * However, this method uses the minimum difference to slowly
50  * approach feasibility, in the hopes of getting a better solution.
51  *
52  * MaxProfit/3:
53  * Make multi-dim feasible based on maximizing profit. This
54  * procedure is more expensive then others becuase it
55  * attempts to gain feasibility while staying at the
56  * highest profit via local search heuristic.
57  *
58  * Add Randomized Teacher mod:
59  *
60  * 0: do not include mod
61  * 1: include mod
62  *
63  * Teacher is chosen at random (uniform distribution) from the top
64  * 10% of a population.
65  *
66  * Add Simulated Annealing mod:
67  *
68  * 0: do not include mod
69  * 1: include threshold annealing.
70  *
71  * Size of Classroom:
72  * Introduced as i-tlbo, the number of teachers is determined by
73  * popluation size / size of classroom. For one teacher this value
74  * should equal zero.
75  *
76  *******************************************************************/
